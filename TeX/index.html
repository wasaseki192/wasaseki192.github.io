LaTeXの環境構築方法

オンラインでの環境構築

  1. Overleafを開きます。
  2. 右上の"Sign up"を押します。
  3. アカウントを作成します。
  4. アカウントにログインできたら左の方の"New project"を押します。
  5. 出てきた中から"Blank project"を押します。
  6. プロジェクトの名前を適当に決め、"Create"を押します。
  7. "main.tex"に記事テンプレートか物理実験テンプレートを貼り付けます。
  8. "File tree"の右にある"New file"を押し、"File Name"を"latexmkrc"として作成する。
  9. latexmkrcをlatexmkrcに貼り付ける。
  10. 左下の"Setting"を押し、"Compiler"から"Compiler"を"LaTeX"にする。
  11. "Setting"から元の画面に戻り、右の方にある"Recompile"を押す。

オフラインでの環境構築

TeX Liveをインストールする

  1. TeXLiveのインストーラーをダウンロードし、実行します。
  2. Windows Defenderに保護されることがありますがファイルに問題はないので、「詳細情報」を押して実行します。
  3. "Install"を選択して"Next"を押します。
  4. もう一度"Next"を押します。
  5. しばらく放置すると「TeX Live インストーラー」というウィンドウが表示されます。
  6. 「TeXworksをインストールする」のチェックを外し、「高度な設定」を押します。
  7. 「選択したもの」の「スキーム」を「basicスキーム（plainおよびlatex）」に変更します。
  8. 「スキーム」の下の「カスタマイズ」を押し、「言語」から「日本語」と「英語・米語」を選択、「ほかのコレクション」から「Latex推奨パッケージ」を追加で選択し、「OK」を押します。
  9. 右下の「インストール」を押します。

TeX Liveのインストールがうまくいかない場合

  1. TeXLiveのインストーラーをダウンロードし、実行します。
  2. Windows Defenderに保護されることがありますがファイルに問題はないので、「詳細情報」を押して実行します。
  3. "Unpack only"を選択して"Next"を押します。
  4. "Browse"を押し、Cドライブの中に"temp_texlive"フォルダーを作り、そのフォルダーを選択します。
  5. "Next"を押します。
  6. "Install"を押します。
  7. 解凍が完了すると"Close"が押せるようになるので"Close"を押します。
  8. 「コマンドプロンプト」を起動します。
  9. "mkdir C:\temp"と入力し実行します。
  10. "set TEMP=C:\temp"と入力し実行します。
  11. "set TMP=C:\temp"と入力し実行します。
  12. "cd C:\temp_texlive"と入力し実行します。
  13. "cd install"と入力し、Tabキーを押して実行します。
  14. "install-tl-windows.bat -no-gui"と入力し実行します。
  15. しばらくするとインストーラーが起動します。
  16. "Enter command"に"S"と入力してEnterキーを押します。続いて"d"と入力してEnterキーを押し、"R"と入力してEnterキーを押します。
  17. "Enter command"に"C"と入力してEnterキーを押します。続いて"p"と入力してEnterキーを押し、"x"と入力してEnterキーを押し、"F"と入力してEnterキーを押し、"R"と入力してEnterキーを押します。
  18. "Enter command"に"I"と入力してEnterキーを押します。
  19. 「TeX Liveへようこそ！」と表示されたらインストール完了です。
  20. "cd C:\"と入力し実行します。
  21. "rmdir /s /q C:\temp"と入力し実行します。
  22. "rmdir /s /q C:\temp_texlive"と入力し実行します。

TeXLiveマネージャーの設定

  1. TLShell TeX Live Managerを起動します。
  2. 「パッケージリスト」の「状態」を「すべて」に変更し、「検索」から"latexmk"を検索します。
  3. "latexmk"と"latexmk.win32"が下に出てくるので左側の丸を押してチェックし、画面右側の「選択項目をインストール」を押します。
  4. 終了したら「閉じる」を押し、「終了」を押します。

VSCodeをインストールする

  1. 使っているOSに合わせてVSCodeをダウンロードします。
  2. ダウンロードしたファイルを実行して、「次へ」を押し続け、「インストール」を押してインストールします。
  3. OSを再起動し、VSCodeを開きます。
  4. 画面左側にある拡張機能（ブロックのマーク）を押します。
  5. "Japanese Language Pack for Visual Studio Code"と"LaTeX Workshop"をインストールします。

latexmkrcの設定

  1. エクスプローラーを開き、上部の「表示」から「隠しファイル」のチェックを付けます。
  2. メモ帳でこのファイルの中身を貼り付けます。
  3. CドライブのUsersフォルダー(あるいはユーザーフォルダー)の中にある自分のユーザー名に対応するフォルダーの中に保存します。

VSCodeの設定

  1. VSCodeを開き、左下の設定（歯車ボタン）を押し、設定を開く。
  2. 設定画面の右上にある設定ボタン（紙をひっくり返すマーク）を押す。
  3. 中括弧の中にこのファイルの中身を貼り付ける。
